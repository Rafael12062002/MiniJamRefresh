English

Hello, 
I will help you implement the asset if you are a beginner. 
To put the template in your project, 
you must first unzip the file and then drag the file in .unitypackage format, 
which already comes standard, so just be careful with this. 
Drag it from your download folder to the root of the project. From this moment on,
you can access the scenes, but to use them, 
you must go to the unity menu and press file -> BuildSettings and with the scene open, add them one by one to the building, put them in order.
Now the next step is very important, initially your scripts will come with an error, because you need to download the Localization dependency in the unity package,
to do this process go to the unity menu in the window -> Package manager option.
In the Unity Registry option look for the Localization dependency and download it, this will eliminate the script errors, it turns out that this dependency is used to translate the game's texts
The game already comes with the menu and intro all configured for English and Portuguese, to add it to the rest of the game you need to put the references in the text localization table, 
in which I will create a video teaching you how to do it, after that everything is already configured and functional, good luck in creating your game :)
